# DaoChai (Web)
 
## Описание проекта
Данный проект представляет собой учебный QA-проект по тестированию веб-версии интернет-магазина DaoChai (https://daochai.ru).

Цель проекта — продемонстрировать навыки ручного тестирования, анализа требований и оформления тестовой документации на примере реального продукта.

## Объект тестирования
Веб-версия интернет-магазина DaoChai.

## Область тестирования (Scope)
- Главная страница
- Корзина

## Вне области тестирования (Out of Scope)
- Авторизация и личный кабинет
- Каталог товаров
- Карточка товара
- Оплата и интеграции с платёжными системами
- Мобильное приложение

## Артефакты тестирования
В рамках проекта подготовлены:
- Чек-листы
- Тест-кейсы
- Баг-репорты

## Тип тестирования
- Ручное функциональное тестирование

## Инструменты
- Google Sheets / Google Docs
- GitHub

## Результат
Проект используется в качестве портфолио для демонстрации практических навыков QA Engineer (Junior).
